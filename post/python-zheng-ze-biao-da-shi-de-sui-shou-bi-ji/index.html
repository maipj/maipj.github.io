python正则表达式的随手笔记

2021-04-01 3 min read

这个东西最好在交互式环境中进行测试

import re 进库
op=re.complie('\d\d\d-\d\d\d-\d\d\d\d')
mo=op.search('我的电话为 444-123-5678') 当然这肯定是假的，
mo.group()
444-123-5678 #这是我们 理所应当 得到的结果
有re.compile( ) 把规格写进库中（就写进程序中等待search中的字符串去比较）
比如 变量 x=re.compile( r'\d\d\d-\d\d\d-\d\d\d\d ') # \d表示的是数字
得到的x可以用search()findall()方法在该变量
search是寻找第一个遇到的，而findall顾名思义寻找search（‘’）中字符串中 的所有符合上面写进re.compile( )规格中的所有。
search( )返回的对象中可以用group( )。

拓展：
利用括号分组：
( r'(\d\d\d)-(\d\d\d-\d\d\d\d)' ) 第一对括号则是group(1):\d\d\d 而第二对括号则是group（2）: \d\d\d-\d\d\d\d 。这里只不过是通过调用gruop（）中的参数那样调用，而groups（）则把所有分组组合成一个元组就像( '\d\d\d','\d\d\d-\d\d\d\d' )

用管道匹配对各分组:
op=re.copile(r'Batman|Teain Fey') 这就像BAtman 或者Teain Fey
跟以上一样，search遇到第一个就返回值，然后再调用group( )；
比如 tt=op.search('Batman Teain Fey')
>>>tt.group()
Batman
反则若search字符串调转过来结果也会相反

           其中还有一个知识点比较重要：可以用口号进行统计管理: compile(r'Bat(man|moblie|copter|bat)')
           .group()即可输出第一个匹配的字符串
           .group(1)即可出现括号里面匹配字符串中的组成成员 











    现在讲一下新内容

不区分大小写的匹配：
也就在用re.compile()中加多一个参数，即re.compile('xxx',re.I)或者re.IGNORECASE
re.compile中的xxx在search中出现大小写都可以返回xxx

用sub（）方法替换字符串：
sub（字符串a，字符串b）字符串a是在字符串b中找到正则表达式中的数据后把该数据替换并返回字符串，字符串a还可以用 r'\1***' 显示第一个字母隐藏后面的

组合使用 大小写 和 读换行符 以及编写注释:
可以在re.compile(1,2) 2号位用 “ | ”管道字符经变量组合起来
